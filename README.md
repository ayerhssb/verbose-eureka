# Portfolio GUI

My portfolio website developed with Next.JS(SSG) and TypeScript. Tailwind CSS and GSAP is used for styling and animations. Light & Dark themes supported.

## Features

- Responsive Design 📱💻
- Light & Dark themes ☀️🌙
- Fully Accessible ♿️

## Tech Stack

**Frontend** - [NextJS](https://nextjs.org/), [React](https://reactjs.org/), [TypeScript](https://www.typescriptlang.org/)  
**Styling** - [Tailwind CSS](https://tailwindcss.com/)  
**Animations** - [GSAP](https://greensock.com/)   
**Deployment** - [Vercel](https://vercel.com/)

## Running Locally

Clone the project

```bash
git clone https://github.com/ayerhssb/verbose-eureka.git
```

Go to the project directory

```bash
cd verbose-eureka
```

Remove remote origin

```bash
git remote remove origin
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev
```
